
# Electron Angular Boilerplate

Electron + photonkit + Angular boilerplate. 

### Angular-cli
Use angular-cli to scaffold your project.
Generate new component, routes... in one command line
```bash
ng g component chat
```

### Run your app
```bash
# ng serve (browser)
yarn run start

# electron serve
yarn run electron 

# package your app
yarn run package-all
```
